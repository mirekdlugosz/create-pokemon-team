Something on this website doesn't work? You see something that you did not expect? You struggle to make sense out of something?

Then you have found a problem!

Please tell us about it:

  • Raise an issue on GitHub tracker. This requires GitHub account (you can create one for free), but is the best way to make your problem visible.
  • Send message to Mirosław Zalewski on Reddit, if you have an account.
  • Tweet to Mirosław Zalewski on Twitter, if you have an account.
  • Email Mirosław Zalewski. Technically it does require email account, but everyone has one anyway.
